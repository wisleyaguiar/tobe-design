CTA button used across TO/BE DASH — for report actions, hero CTAs, table row actions.

```jsx
<Button variant="primary" size="md" onClick={() => {}}>Gerar relatório</Button>
```

Variants: `primary` (solid red, main actions), `secondary` (outlined, mid-gray border), `ghost` (text-only, tertiary). Sizes: `sm` / `md` / `lg`. Never pill-shaped — radius is 4px max, per brand (no rounded, no bounce on press).
