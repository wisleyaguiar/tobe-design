# DESIGN SYSTEM — TO BE DASH

## Filosofia Visual

A plataforma deve transmitir:

- luxo
- performance
- sofisticação
- branding premium
- editorial fashion aesthetics
- dashboards cinematográficos

O visual deve parecer uma mistura entre:

- Bloomberg Terminal moderno
- Revista editorial premium
- Dashboard de marca de luxo
- Interface de agência high-end

Nunca parecer:

- ERP genérico
- admin dashboard bootstrap
- template SaaS comum
- sistema corporativo antigo
